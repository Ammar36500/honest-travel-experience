# Honest Travel Review

A web-based application that offers free and authentic reviews of travel experiences.

## Description

A web-based Honest Travel Experience Platform that aims to solve the problem of sponsored and biased content on travel review websites. 
I am committed to offer passengers free and authentic reviews of travel experiences that they can utilize to make informed decisions.

## Features

- User account and profile management
- Review submission with ratings and images
- Searchable location database
- Multi-tiered review moderation
- Interactive review and rating visualization
- Community feedback
- AI-powered travel assistant

## Installation

1. Clone this repository
   ```bash
   git clone <https://github.com/Ammar36500/honest-travel-experience>
   ```

2. Navigate to the project directory:

   cd honest-travel-review

3.Install the required dependencies:

pip install -r requirements.txt
   
4. Run the development server:

python manage.py runserver

5. Open your web browser and go to http://localhost:8000
